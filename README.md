### Loyihada o'rganilgan yangi teglar:

* **HTML Tables (Jadvallar):** Ma'lumotlarni qator va ustunlarga bo'lib, tartibli joylashtirishni o'rgandim. Jadval ichida qatorlar yaratish uchun `<tr>`, sarlavhalar uchun `<th>` va oddiy kataklar uchun `<td>` teglaridan foydalanildi.
* **HTML Marquee (Yuruvchi matn):** Saytdagi matnlarga harakatlanish effektini berishni sinab ko'rdim. `<marquee>` tegi yordamida matnni o'ngdan chapga yurgizish va uning tezligini sozlashni o'rgandim.
